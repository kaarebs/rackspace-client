Rackspace client

Add it to composer

"require": {
        "clauswitt/rackspace-client": "dev-master"
    }

and use it like this:

require('vendor/autoload.php');
$client = new \ClausWitt\Rackspace\Client\Server\Client('rackspacecloudusername', 'apikey');

$flavours = $client->getFlavours();
$images = $client->getImages();

print_r($client->createServer('test', $images[0]->id , $flavours[0]->id));

This will create a new server on rackspace cloud with the name test, and the first available image and flavour attached.

The following methods have been implemented:

  • createServer($name, $imageId, $flavourId)
  • getServers()
  • getServer($id)
  • getFlavours()
  • getFlavour($id)
  • getImages()
  • getImages($id)
  • getServerFromIp($ip)
  • getServerIpsMatching($name)
  • getServerCountMatching($name)
  • getServersMatching($name)
